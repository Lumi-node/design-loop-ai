# Installation

## From PyPI

```bash
pip install design-loop-ai
```

## From Source

```bash
git clone https://github.com/Lumi-node/design-loop-ai.git
cd design-loop-ai
pip install -e ".[dev]"
```

## Requirements

- Python >= 3.10
- Dependencies: agent, color_extractor, colorsys, design_agent, html, image_loader, layout_detector, main
