# Installation

## From PyPI

```bash
pip install design-loop-ai
```

## From Source

```bash
git clone https://github.com/Lumi-node/design-loop-ai.git
cd design-loop-ai
pip install -e ".[dev]"
```

## Requirements

- Python >= 3.10
- Dependencies: agent_designer, bs4, html, html_generator, importlib, metrics
